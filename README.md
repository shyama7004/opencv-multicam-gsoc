# GSoC 2025 – OpenCV Multi-camera Calibration

This repository collects the Pull Requests I contributed during Google Summer of Code 2025 under the **OpenCV** organization.  
The work focused on **multi-camera calibration tools**, including dataset preparation, visualization, and documentation.
